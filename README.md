# Introduction

Evmos blockexplorer querying and indexing EVM transactions and giving insight in EVM chain health. Also includes several validator insights.

# Located at

`explorer.evmos.kryha.io`

# Uses and requires

- NodeJS 14+
- ReactJS
- TypeScript
- MongoDB

# Running locally

in a terminal:
start a MongoDB server

in a new terminal:

```
cd backend
yarn install
yarn start
```

in a new terminal:

```
cd frontend
yarn install
yarn start
```
